Bienvenidos a nuestro Proyecto

Esta aplicación ha sido diseñada en Visual Basic .NET para facilitar la gestión de archivos. Es una herramienta ligera y rápida para el día a día.

Objetivos del Proyecto

Nuestro objetivo principal es ofrecer un conversor de archivos y un sistema de compresión seguro, todo bajo una interfaz moderna y fácil de usar para cualquier usuario.

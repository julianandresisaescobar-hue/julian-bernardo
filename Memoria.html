Estructura Técnica

Nuestro proyecto ha sido desarrollado utilizando el entorno de Visual Basic .NET. Hemos elegido esta tecnología por su robustez a la hora de gestionar archivos del sistema y su facilidad para crear interfaces de usuario profesionales.

Funcionalidades del Sistema

El programa permite gestionar archivos de manera inteligente. Hemos incluido un motor de conversión que transforma datos entre formatos TXT y PDF sin perder información, facilitando el trabajo diario del usuario.

Además, el sistema cuenta con una opción de compresión de datos para ahorrar espacio en el disco duro, optimizando los recursos del ordenador al máximo.

Lógica del Menú Dinámico

Para aprovechar el espacio, el menú lateral puede hacerse pequeño o grande según lo que necesite el usuario. Esto se consigue con una instrucción de código muy sencilla que controla el ancho del panel:

If panelMenu.Width > 50 Then panelMenu.Width = 50 Else panelMenu.Width = 200

Personalización de Colores

Sabemos que cada usuario es diferente, por eso hemos añadido un sistema de temas. Se pueden cambiar los colores de la aplicación para que sea más cómoda de usar por la noche o en ambientes con mucha luz, evitando el cansancio visual.

Conclusión y Futuro

Actualmente el proyecto está en su fase final y funciona perfectamente. En el futuro, nos gustaría añadir más idiomas y la posibilidad de conectar el programa con una base de datos en la nube.
